# Tab Bar Icons

## Required Icons

You need to download and place the following icons in this folder:

### 1. Home Icon
- **Files needed:**
  - `home.png` (25x25px @1x)
  - `home@2x.png` (50x50px @2x)
  - `home@3x.png` (75x75px @3x)
- **Search terms:** "home icon", "house icon"

### 2. Explore Icon

### 3. Go Live Icon

## Icon Requirements

- **Format:** PNG with transparency
- **Color:** Single color (black or white) with alpha channel
- **Style:** Solid/filled style (iOS automatically handles gray/tint colors)
- **No drop shadows**
- **Use anti-aliasing**
